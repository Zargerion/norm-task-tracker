# NORM Task Tracker

Внутренний таск-трекер с проектами, материалами, ролями и изолированными **пространствами** (multi-tenant). Авторизация через сайт и Telegram-бота: регистрация, подтверждение в боте, magic link и вход по логину/паролю. Стек: **NestJS**, **Prisma**, **PostgreSQL**, **Next.js** (App Router), **grammy**, JWT в httpOnly-cookie, деплой через Docker Compose и Ansible.

## Зачем этот репозиторий

Проект написан **ради интереса**, код целиком сгенерирован и доведён до рабочего состояния **Claude** (AI от Anthropic), без «ручного» написания с нуля человеком. На выходе получился довольно цельный продукт: API, фронт, бот, схема БД, сиды, документация в `docs/` — отсюда и ощущение «удивительности»: не игрушечный todo, а полноценная заготовка под реальную эксплуатацию с ролями, rate limiting и своей дизайн-системой.
